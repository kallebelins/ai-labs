# Lab 77: 77 Chatbot Reclamacoes Publicas

## Problema a ser resolvido

Chatbot para receber, categorizar e priorizar reclamações do cidadão (prefeitura, órgãos públicos).

## Métricas de teste detalhadas

### Métricas principais
- **Tempo de resposta, taxa de resolução, priorização correta.**: Métrica específica do laboratório

### Métricas adicionais
- **Performance**: Tempo de resposta, throughput, uso de recursos
- **Qualidade**: Precisão, recall, satisfação do usuário
- **Escalabilidade**: Comportamento sob carga, crescimento de dados
- **Manutenibilidade**: Facilidade de manutenção e extensão
